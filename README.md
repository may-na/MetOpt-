# 3. Приведение задачи к каноническому виду

### Ограничение 1 (≤):
Добавляем slack-переменную \(s_1\):

\[
x_1 + x_2 + 2x_3 + s_1 = 6
\]

### Ограничение 2 (=):
Добавляем искусственную переменную \(a_1\):

\[
x_2 + x_3 + x_4 + a_1 = 4
\]

### Ограничение 3 (≥):
Добавляем избыточную переменную \(s_2\) и искусственную \(a_2\):

\[
2x_1 + x_4 - s_2 + a_2 = 3
\]

### Каноническая форма:

\[
\begin{cases}
x_1 + x_2 + 2x_3 + s_1 = 6,\\
x_2 + x_3 + x_4 + a_1 = 4,\\
2x_1 + x_4 - s_2 + a_2 = 3,\\
x, s, a \ge 0.
\end{cases}
\]

# 4. Вспомогательная задача (Фаза I)

\[

\]

\[

\]

→ Допустимое решение существует, можно переходить ко **Фазе II**.
